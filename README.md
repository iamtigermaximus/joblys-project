
## Getting Started

These instructions will help you clone and set up the project on your local machine.

### Prerequisites
- Node.js 
- npm 
- Git

Frontend stack:
- NextJS
- Typescript
- styled-components
- Next-Auth

### Clone the Repository

Run the following command:
1. git clone https://github.com/iamtigermaximus/joblys-project.git
2. cd joblys-project
3. npm install
4. npm run dev
